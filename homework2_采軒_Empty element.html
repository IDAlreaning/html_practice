Empty element 置空元素

甚麼是Empty element(置空元素)？ 

    1. HTML elements with no content are called empty elements.
    
       EX.<br>是一個沒有結束標記的置空元素（<br>定義換行符號）。
    
    2. 置空元素可以在開始標籤中被關閉，如：<br />。

舉例時間

【第一個】<br>

Inserts a single line break.

    程式碼：
    <h1>This is heading 1</h1>
    <p>This is some text.</p>
    <br>
    <h2>This is heading 2</h2>
    <p>This is some other text.</p>
    <br>

【第二個】<hr>

在HTML頁面中顯示出水平線。

    
    程式碼：      
    <h1>This is heading 1</h1>
    <p>This is some text.</p>
    <hr>
    <h2>This is heading 2</h2>
    <p>This is some other text.</p>
    <hr>

【第三個】<img>

圖像可以改進網頁的設計和外觀。

    
    程式碼：
    <img src="https://www.google.com.tw/images/branding/googlelogo/2x/googlelogo_color_272x92dp.png" alt="google">
